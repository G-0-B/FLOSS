# FLOSSI0ULLK Radicle Stack

Keep the stack legible:

- `Radicle` is the code collaboration substrate.
- `Local source_chain + MCP` is the immediate agent bridge.
- `Holochain` is the provenance and validation substrate.
- `GitHub` is optional mirror space, not the architectural center.

## Core workflow

1. Read the current architecture and ADR surfaces before editing.
2. If the substrate model changes, update architecture docs and the ADR index in the same pass.
3. Preserve the distinction between dev-plane collaboration and runtime-plane provenance.
