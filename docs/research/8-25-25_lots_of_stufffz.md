## Minimal reference implementation (Rust): “TrustWeave” attestation bus

```rust
//! TrustWeave: a tiny, composable attestation bus for YumeiCHAIN.
//!
//! Philosophy: every contribution (fact, model delta, norm decision) is a signed promise
//! to our commons. We treat strong typing as integrity, error handling as humility,
//! and logging/observability as our collective learning loop.

use std::time::SystemTime;
use thiserror::Error;
use serde::{Deserialize, Serialize};
use tracing::{info, warn, instrument};

/// A DID, DID-URL, or other stable agent identity.
pub type AgentId = String;

/// Content-addressed reference (e.g., IPFS CID).
pub type Cid = String;

/// Cryptographic signature bytes (detached), algorithm in `alg`.
#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct Signature {
    pub alg: String,
    pub sig_b64: String,
    pub key_id: String, // e.g., OpenPubKey bound to OIDC subject
}

/// Minimal attestable statement.
#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct Attestation {
    pub subject: AgentId,        // who this is about
    pub predicate: String,       // e.g., "created:artifact", "trained:delta", "vouched:claim"
    pub object: Cid,             // what (content-addressed)
    pub issued_at: SystemTime,
    pub issuer: AgentId,         // who signs
    pub signature: Signature,
    pub context: serde_json::Value, // freeform: policy tags, metrics, etc.
}

/// Errors reflect reality; we surface them clearly and recover gracefully.
#[derive(Error, Debug)]
pub enum TrustError {
    #[error("policy violation: {0}")]
    Policy(String),
    #[error("crypto failure: {0}")]
    Crypto(String),
    #[error("storage error: {0}")]
    Storage(String),
    #[error("transient: {0}")]
    Transient(String),
}

/// Policy engine contract (e.g., OPA/Rego WASM, rule graph).
pub trait PolicyEngine: Send + Sync {
    /// Return Ok(()) if allowed; Err(Policy(...)) otherwise.
    fn evaluate(&self, att: &Attestation) -> Result<(), TrustError>;
}

/// Storage backends are pluggable (Holochain zome call, Rekor, IPFS pin, etc.).
#[async_trait::async_trait]
pub trait AttestationStore: Send + Sync {
    async fn publish(&self, att: &Attestation) -> Result<(), TrustError>;
    async fn verify(&self, att: &Attestation) -> Result<(), TrustError>;
}

/// The bus composes policy + many stores (fan-out write, quorum verify).
pub struct TrustWeave<E: PolicyEngine> {
    engine: E,
    stores: Vec<Box<dyn AttestationStore>>,
}

impl<E: PolicyEngine> TrustWeave<E> {
    pub fn new(engine: E, stores: Vec<Box<dyn AttestationStore>>) -> Self {
        Self { engine, stores }
    }

    /// Validate → sign/verify → publish everywhere we’re configured.
    #[instrument(skip(self, att))]
    pub async fn commit(&self, att: &Attestation) -> Result<(), TrustError> {
        self.engine.evaluate(att)?;
        let mut ok = 0usize;
        for s in &self.stores {
            match s.publish(att).await {
                Ok(()) => ok += 1,
                Err(e) => warn!(error=?e, "store publish failed"),
            }
        }
        if ok == 0 {
            return Err(TrustError::Storage("no backends accepted attestation".into()));
        }
        info!(accepted_backends = ok, "attestation committed");
        Ok(())
    }
}
```

4. **Ethics** – Indigenous values guide evaluation of harms, benefits, and future use of data.
