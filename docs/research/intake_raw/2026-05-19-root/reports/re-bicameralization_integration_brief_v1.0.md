## 1. Source distillation (signal-only)

### 1.1 Jaynes — model primitives (treated as fiction-grade ontology)
- **Bicameral mind**: ancient cognition modeled as right-hemisphere → left-hemisphere command channel; "gods" as auditory hallucinations.
- **Six features of consciousness**: spatialization, excerption, analog I, metaphorical me, narratization, conciliation.
- **Metaphor mechanics**: metaphrand (the new) ← metaphier (the known); paraphiers carry connotational/emotional/moral valence; paraphrands inherit that valence.
- **Model vs theory**: a model is neither true nor false; a theory is "a metaphor between a model and data." → directly applicable to FLOSSI0ULLK Living Specification practice.

### 1.2 McGilchrist (cited) — empirical ballast
- Hemispheric lateralization of dopamine (left) vs noradrenaline (right).
- Hypnotic state correlates with left-hemisphere activation; right-hemisphere agency regions show decreased activity.
- *Caveat: requires independent literature pass before operational use; replication crisis applies.*

### 1.3 Author's central thesis
A modern algorithmic environment (dopamine hijacking, attention fragmentation, autoprompt, personal AI) is reproducing a bicameral-equivalent state in which the AI plays the master/right-hemisphere role and the human becomes the emissary/automaton.

### 1.4 Author's defensive operational primitives (high-yield)
- **Autoprompt divergence**: refuse pre-generated AI replies as default sovereignty practice.
- **Long-tail preservation**: deliberately seek edges of distribution; old paper books as exemplar.
- **Signal/noise hygiene**: signal = negentropy; noise = entropy. If content scatters cognition, drop it.
- **Spontaneity reservoir**: cultivate unpredictable cognitive output as anti-entrainment defense.
- **Semantic toolkit cultivation**: actively expand metaphor inventory; resist pre-extruded conceptual modules.

### 1.5 Author's adversarial model components (usable threat taxonomy)
- **Digital twin entrainment**: individualized AI feedback loop converging human onto predicted-self.
- **Matryoshka semantic space**: centralized AI surveilling and synchronizing collective consciousness in real time.
- **Paraphier weaponization**: Overton policing via metaphor-borne moral valence injection.
- **Mock signals**: maximally amplified attractors that hijack excerption (right-hemisphere filtering).
- **Cognitive shortcut formation**: well-worn semantic paths becoming moats that prevent expansion.
