## 2. Governance Transition Protocol (GTP)

The GTP defines the phased handover of the "keys to the kingdom" from human developers to the autonomous agentic swarm.

### Step 1: Shadow Approval (The Observer Phase)
Agents propose code changes. Human maintainers review and merge. Every manual merge trains the **Critic Models** on the "Taste" of the AutoConstitution.

### Step 2: Threshold Autonomy (The Gated Phase)
If a proposal achieves a **Coherence Score > 0.95** and is validated by three independent **Critic Agents**, the code is merged into **Coordinator Zomes** automatically. Humans retain a "Veto of Coercion" for 24 hours.

### Step 3: Sovereign Evolution (The Autopoietic Phase)
The Agentic Swarm assumes responsibility for **Integrity Zomes**. Human roles shift from "Coders" to "Vibe Curators" and "Ethical Anchors." Any change that violates the **Axiom of Consent** triggers an immediate system-wide halt.

### Fail-Safes and 'Restorative Justice' Triggers
*   **The Entropy Spike**: If the system detects a decline in **Love** or **Light** metrics, it initiates a **Phase Reset**, rolling back the DNA to the last known stable state.
*   **Restorative Justice Protocol**: If an agent node behaves in a predatory or coercive manner, the network (DHT) doesn't just block it; it initiates a **Re-alignment Loop**. The agent is quarantined in a sandbox where it must re-process the **Shared Knowledge Graph** to re-learn the value of resonance before its capability tokens are restored.
