## Authority tier system (LSM-Override is the key constraint)

| Tier | Definition | LLM default? |
|---|---|---|
| `[auth:lived]` | Born/raised; 10+ years embodied | No (cannot be assigned to LLM) |
| `[auth:trained]` | Formally studied; can derive | Possible with community consent |
| `[auth:structural]` | Articulate axioms; derive conclusions | YES — default for all LLMs |
| `[auth:tourist]` | Recognize but not analyze | Default for new participants |

LSM-Override protocol: `[auth:trained]` or `[auth:lived]` challenge to an LLM `[auth:structural]` claim automatically flags `cfis:contested` and routes to frame-community deliberation. Prevents sophisticated LLM argumentation from colonizing frame spaces.
