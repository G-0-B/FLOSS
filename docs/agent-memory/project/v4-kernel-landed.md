## The orthogonal-axis claim

CCES (HOLISTIC_ARCHITECTURE.md §2.5) is the **teleological architecture** — what flourishing is served (L0 Cosmological Telos through L7 AI Moral Subjects). v4.0 Kernel is the **operational architecture** — what agents do, how artifacts flow, where authority sits, what is forbidden (Layer -1 Telos through Layer 8 Pilot Holon).

They constrain each other; they don't replace each other. Future work should cite both axes when relevant.
