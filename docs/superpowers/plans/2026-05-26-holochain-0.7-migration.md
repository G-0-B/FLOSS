## Why this plan exists

`hc 0.6.1` (the version `holonix main-0.6` pins) is **EOL upstream** ‚Äî `crates.io` latest `holochain_cli` is `0.7.0-dev.26` as of 2026-05-26 ([5969d09](.) verification). No `@holochain/tryorama` version pairs cleanly with hc 0.6.1, so the **ADR-12 end-to-end consent_gate.test.ts claim** AND the **MVP_PLAN Phase 0 Tryorama claim** are currently `NOT verified` per their honest truth-status corrections.

Migrating to `holonix main-0.7-dev` makes `tryorama 0.18+` pair correctly with the conductor, closing both evidence gaps.
