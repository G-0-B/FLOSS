# Archived Documentation

This directory contains archived versions of documentation files that have been superseded by canonical versions.

## Archived Master Specification Drafts

The following files are earlier drafts of the master specification. The canonical version is now at `../arf_flossi_0_ullk_sdd_master_specification.md` (v0.1):

- `arf_flossi_0_ullk_sdd_master_specification_v_0.md` - Initial draft
- `arf_flossi_0_ullk_sdd_master_specification_v_00.md` - Early draft
- `arf_flossi_0_ullk_sdd_master_specification_v_000.md` - Early draft

All three contained nearly identical content with minor formatting differences. They were archived to reduce confusion about which specification is authoritative.

## Archive Policy

Files are moved here rather than deleted to preserve history and provenance. The confusing version numbering scheme (v_0, v_00, v_000, v_01) was a result of iterative development without a clear versioning convention.
