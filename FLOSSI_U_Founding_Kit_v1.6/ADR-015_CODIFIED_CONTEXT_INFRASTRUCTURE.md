# ADR-015 — Codified Context Infrastructure

**Status:** Accepted
**Date:** February 24 2026

**Context:** "Codified Context: Infrastructure for AI Agents in a Complex Codebase" defines tiered hot/cold memory, specialist agents, and an MCP retrieval server.

**Decision:** Adopt as memory & knowledge infrastructure.
- Hot constitution (always-available meta-rules).
- Specialist agents (cognition).
- Cold KB (archived Knowledge Triples).
- MCP retrieval server for external AI tools.

**Consequences:**
- Enables scalable, verifiable memory across the swarm.
- Facilitates MCP-based queries from external models.
